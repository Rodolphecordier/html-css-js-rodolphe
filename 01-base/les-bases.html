les balises DIV :

la balise DIV, est une balise neutre (aucun style par défaut) de type block, elle permet de structurer la page

les balises block agissent sur un axe vertical et donc génère un retour a la ligne.
je pourrais toutes fois changer leurs comportement avec du CSS.

les balises Inlineelles, agissent sur un axe horizontal et donc ne génère aucun retour a la ligne.
je pourrais toute fois changer leurs comportement avec du CSS.

les formulaires :

  • la balise form permet de créer un formulaire
  • name:permet de donner un nom au formulaire
  • action cet attribut est OBLIGATOIRE et permet de renseigner le script / la page vers laquelle l'utilisateur sera redirigé lors de l'envoi du formulaire cette page va recevoir l'ensemble des données saisies par mon utilisateur.
  • method permet de préciser quelle methode [get ou post] doit être utilisée par le navigateur pour transmettre les données saisies par l'utilisateur vers la page action














créer un lien en HTML

  • la balise "a" permetde créer un lien vers une autre resource.(images,son,videos, PDF...

  • l'attribut HREF : permet de définir le chemin vers le fichier à lier

  • l'attribut TARGET : prend plusieurs valeurs: "_blank" pour ouvrir un lien dans une nouvelle fenetre par exemple...

  • l'attribut TITLE : permet de donner un titre au lien.ce titre sera afficher dans une bulle au survol du line par la sourie
